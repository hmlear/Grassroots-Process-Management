# Grassroots Process Management – Online Appendix

This repository contains the online appendix accompanying the paper:

> *Grassroots Process Management*.

## Repository Contents

- `Grassroots_Projects.pdf` — Full case documentation for the 26 citizen-led initiatives analysed in the paper, organised by archetype and outcome (successful/failed)

## Case Selection

Cases were identified using a snowball-inspired, criteria-based strategy.
To be included, cases were required to:

1. Be initiated by citizens, communities, or civil society actors
2. Exhibit recurring process enactment over time
3. Pursue public or collective value rather than commercial benefit
4. Provide sufficient documentation to reconstruct the process and governance evolution

ChatGPT was used as a supplementary search tool; all suggestions were manually verified, and no case was included solely on that basis.

## Archetypes

Cases are organised into four archetypes:

- **Community Knowledge Builders** — citizens collectively producing previously unavailable information to fill visibility gaps, which institutions had ignored
- **Civic Infrastructure Creators** — citizens building physical or digital infrastructure in place of institutional provision
- **Participatory Policy Innovators** — citizens designing collective decision-making platforms or methods that governments may later adopt
- **Tactical Mobilisers** — citizens initiating rapid, action-oriented interventions to demonstrate need or pressure institutions
